The proposed research followed the below methodology steps in order to accomplish the expected outcome.

Library Installation

The first step is to install the necessary Python libraries. These libraries provide essential tools and functions for image processing, machine learning, and computer vision. commands to install the “pystan” and “prophet” libraries using pip. These libraries are used for time series forecasting and trend analysis. essential to ensure that we have the correct versions of these libraries installed.

Import Libraries

Once the libraries are installed, import them at the beginning of the Python script. below’s what each library does:

• import pandas as pd: Pandas is a powerful data manipulation library that allows us to work with structured data (e.g., CSV files, dataframes).

• import matplotlib.pyplot as plt: Matplotlib is a plotting library used for creating visualizations, including line plots, scatter plots, and histograms.

• import seaborn as sns: Seaborn is another data visualization library that provides a high-level interface for creating attractive statistical graphics.

• from fbprophet import Prophet: Prophet is a time series forecasting library developed by Facebook. It’s particularly useful for predicting trends and seasonality in data.

Data preprocessing and remove null values

Before training the machine learning model, preprocess the data. preprocessing steps include:

• Analyzing Null values to increase model robustness.

Select relevant features and change index to date

Selecting relevant features involves with the most important ones that will help to the accuracy of our model. I removed Irrelevant or redundant features to simplify the model and improve its performance. Many datasets, especially those involving time series data, include a date or time column. I changed the index to date means setting the dataframe index to this date column, which can be useful for time-based operations and analyses.

Create dataframe and rename columns

Constructed a dataframe from raw data which is a csv file and then renamed the columns to more meaningful names.

Train model and predict future

Involved using historical time-series data to build a forecasting model. Passed API key to fbprophet API for validation. Fit the model for the training data.

Save model and make predictions

Saving a model means writing the trained model to disk so it can be loaded and used later. This is useful for deploying the model in production or sharing it with others.
